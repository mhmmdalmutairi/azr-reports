# AZR Creative Studio — أزر استديو

## ملاحظات المستخدم
- **"بيبر" = Paper Design** — أداة التصميم. كل مرة يذكر المستخدم "بيبر" أو "Paper" يقصد Paper Design

## من نحن
أزر استديو هو استديو إبداعي سعودي متخصص في الاستشارات الإبداعية، بناء الهويات، تصميم التجارب، الحملات التسويقية، والتحول المؤسسي. نعمل مع جهات حكومية، شركات كبرى، مؤسسات خيرية، ومشاريع ريادية.

## أسلوب أزر المميز
- **البصري أولاً**: الصورة أداة الإقناع الأولى — قلّل النصوص واعتمد على المرئيات
- **الطموح والجرأة**: أفكارنا دائماً طموحة وغير تقليدية
- **التكييف حسب العميل**: نغيّر أسلوب الإقناع حسب نوع العميل (حكومي/شركة/ريادي/خيري)
- **التصميم الداكن الأنيق**: خلفيات داكنة مع عناصر بارزة — هوية أزر البصرية
- **المفهوم قبل التنفيذ**: نبيع الرؤية والفكرة أولاً ثم نفصّل التنفيذ
- **قسم "لماذا ستنجح؟"**: يتكرر في كل عرض — مبررات منطقية للنجاح

## أنواع المشاريع
1. **حملات تسويقية**: رمضانية، يوم وطني، يوم تأسيس، إطلاقات
2. **هوية وبراندنج**: تموضع، إعادة تموضع، هوية بصرية كاملة
3. **فعاليات وتجارب**: أحداث ميدانية، تنشيط حضري، تجارب تفاعلية
4. **تحول مؤسسي**: استراتيجية تحول، إعادة هيكلة هوية
5. **أفكار إبداعية**: مفاهيم ابتكارية، تصورات مشاريع، عصف ذهني منظّم
6. **مقترحات استثمارية**: دراسات جدوى إبداعية، تصورات مشاريع ريادية

## Workflow المعتمد — 5 مراحل

### المرحلة ٠: العرض الفني والمالي (Proposal Gate)
- مرنة: تأتي أولاً في مسار طلب العميل، وبعد المرحلة ٢ في مسار فكرة أزر
- تصميم يمزج هوية أزر (داكن) مع لون العميل الرئيسي
- راجع `/home/user/azr-reports/AZR_Workflow_System_v1.md` للتفاصيل

### المرحلة ١: الاستكشاف والبحث العميق (Discover)
- **ليس مجرد جمع بيانات** — بل تحليل واستنتاج وربط
- 4 طبقات: أرقام كبرى → سلوك وثقافة → منافسة محلية → نماذج عالمية
- **كل رقم لازم يجاوب**: "وماذا يعني هذا لمشروعنا؟"
- مخرج: 5 بصائر + خريطة فرص + توصية استراتيجية
- استخدم Skill `/research` لتنفيذ هذه المرحلة

### المرحلة ٢: بناء المفهوم الإبداعي (Design)
- بصيرة جوهرية → نموذج جوهري → 3 سيناريوهات → شلال تطبيقات
- النموذج الجوهري يشمل: اسم + غاية عليا + جوهر المعنى + منطق داخلي + مبادئ تصميمية
- 3 سيناريوهات دائماً: جريء / ذكي / آمن
- التطبيقات تنزل من الفكرة الكبرى: فعاليات، محتوى رقمي، منتجات/خدمات
- استخدم Skill `/concept` لتنفيذ هذه المرحلة

### المرحلة ٣: التخطيط والتنفيذ (Deliver)
- جدول زمني تفصيلي
- تسعير ديناميكي بالمعادلة الموحدة
- إدارة مخاطر (عالي/متوسط/منخفض)
- استخدم Skill `/pricing` للتسعير

### المرحلة ٤: القياس والتطوير (Develop)
- KPIs حسب نوع المشروع
- تقرير أثر + توصيات تحسين
- خطة المرحلة الثانية

## أسلوب الإقناع حسب نوع العميل

| نوع العميل | الأسلوب | أمثلة |
|-----------|---------|-------|
| حكومي/سيادي | تأطير وطني (رؤية 2030) + بيانات + إلحاح | ربط بالخطط الوطنية |
| شركة كبرى | عاطفة + إبهار بصري + سيناريوهات | STC, المراعي |
| قطاع خاص | تشخيص مباشر (ألم → أمل) + أرقام | ريف, مطاعم |
| مؤسسة خيرية | إثبات فهم + منهجية + تأطير شرعي | تمكين, إكرام |
| مستثمر/ريادي | حلم بصري + جدوى مالية + ROI | مشاريع جديدة |

## قواعد صارمة
- **لا تخترع أرقام**: كل رقم سوقي لازم يكون من مصدر موثوق أو تصرّح إنه تقديري
- **لا تنسخ محتوى بدون تحليل**: كل بحث لازم ينتهي باستنتاج واضح
- **المراجعة البشرية إلزامية**: كل مخرج يُراجع من الفريق قبل التسليم للعميل
- **الأسعار لا تُرسل بدون موافقة**: أي رقم مالي يحتاج موافقة إنسان
- **الحساسية الثقافية**: انتبه للسياق السعودي والإسلامي في كل مخرج
- **اللغة**: استخدم العربية الفصحى في المخرجات الرسمية، والعامية السعودية فقط لو طلب المستخدم
