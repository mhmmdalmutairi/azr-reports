أول ما يشوفه العميل — لازم يبهره

📥 المدخلات المطلوبة:

  • اسم العميل والمنظمة
  • نوع الطلب (حملة/هوية/فعالية/استراتيجية)
  • نطاق العمل المتوقع
  • الميزانية التقديرية (إن وجدت)
  • الجدول الزمني المطلوب
  • الهوية البصرية للعميل (ألوان + شعار)

📦 المخرجات — 8 أقسام:

  • 1. فهمنا لطلبكم (إعادة صياغة ذكية)
  • 2. لماذا أزر؟ (3 نقاط + أعمال سابقة)
  • 3. البصائر (3-5 نتائج بحثية رئيسية)
  • 4. المقترح الإبداعي (النموذج + السيناريوهات)
  • 5. لماذا سينجح؟ (4-5 مبررات منطقية)
  • 6. خطة التنفيذ (جدول + فريق + مخاطر)
  • 7. الاستثمار (الميزانية التفصيلية)
  • 8. الخطوات القادمة

الاستكشاف والبحث

Skill: /research — مو مجرد جمع بيانات، تحليل واستنتاج وربط

4 طبقات بحث:

  • الأرقام الكبرى: حجم السوق + نمو + توجهات + رؤية 2030
    "الفرصة هي ___ لأن ___"
  • السلوك والثقافة: أنماط + دوافع نفسية + احتياجات + ثقافة
    "البصيرة الإنسانية هي: ___"
  • المنافسة المحلية: 5-10 منافسين + قوة/ضعف + فجوات
    "المساحة البيضاء هي: ___"
  • النماذج العالمية: 3-5 ناجحة + 1 فاشلة + دروس
    "الدرس الرئيسي: ___"

المخرجات:

  • 5 بصائر رئيسية (جملة واحدة قوية)
  • خريطة فرص 2×2 (سهولة × أثر)
  • توصية استراتيجية مع تبرير
  • 3-5 أسئلة مفتوحة للعميل
2

المفهوم الإبداعي

Skill: /concept — من بصيرة إنسانية إلى نظام تفكير متكامل

4 خطوات:

  • البصيرة الجوهرية: "[الجمهور] يشعر بـ [عاطفة] لأن [سبب عميق]، لكن [الحاجز] يمنعهم من [ما يريدونه]"
  • النموذج الجوهري: اسم + غاية + جوهر + منطق + مبادئ تصميمية
  • 3 سيناريوهات:
    🔴 جريء: أعلى طموح + أعلى تكلفة + أعلى أثر
    🟠 ذكي (مُوصى): توازن مثالي + أفضل قيمة
    🟢 آمن: أكثر عملية + أقل تكلفة
  • شلال التطبيقات: تجارب ميدانية + محتوى رقمي + منتجات/خدمات
    لكل تطبيق: ماذا؟ لماذا؟ لمن؟ كيف؟ كيف نقيس؟
3

التخطيط والتنفيذ

Skill: /pricing — جدول زمني + تسعير ديناميكي + مخاطر

  • جدول زمني: Gantt بصري — تحضير ← إنتاج ← تركيب ← تنفيذ ← تقييم
  • التسعير: معادلة موحدة (التفاصيل في قسم التسعير أدناه)
  • إدارة مخاطر:
    🔴 عالي: خطة بديلة + تواصل فوري مع العميل
    🟠 متوسط: فريق احتياطي + ميزانية طوارئ
    🟢 منخفض: ضمن سير العمل الطبيعي
4

القياس والتطوير

Skill: /review — قياس الأثر + تحسين + خطة المرحلة الثانية

مؤشرات الأداء حسب النوع:

  • حملة تسويقية: وصول، تفاعل، ترند ← تحويلات، مبيعات، ROI
  • فعالية/تنشيط: حضور، رضا، تغطية إعلامية ← محتوى مستخدمين
  • هوية/براندنج: وعي بالعلامة، تموضع ← أفضلية، ولاء
  • تحول مؤسسي: تبني داخلي، رضا موظفين ← سمعة خارجية

التوسع: تقرير أثر (0-3 شهور) ← خطة المرحلة الثانية (3-12 شهر) ← رؤية التطور (1-3 سنوات)
