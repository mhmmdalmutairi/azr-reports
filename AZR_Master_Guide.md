### المرحلة ٠ — العرض الفني والمالي `/proposal`

**المدخلات:** اسم العميل، نوع الطلب، النطاق، الميزانية التقديرية، الجدول الزمني، هوية العميل البصرية

**المخرجات:** عرض من 8 أقسام:
1. فهمنا لطلبكم
2. لماذا أزر؟ (3 نقاط + أعمال سابقة)
3. البصائر (3-5 نتائج بحثية)
4. المقترح الإبداعي (نموذج + سيناريوهات + تطبيقات)
5. لماذا سينجح؟ (4-5 مبررات)
6. خطة التنفيذ (جدول + فريق + مخاطر)
7. الاستثمار (الميزانية)
8. الخطوات القادمة

**التخصيص حسب العميل:**

| نوع العميل | الأسلوب |
|-----------|---------|
| حكومي | لغة رسمية + رؤية 2030 + أثر وطني |
| شركة كبرى | ROI واضح + نماذج عالمية + أرقام أداء |
| قطاع خاص | تشخيص مباشر (ألم → حل) + نتائج ملموسة |
| خيري | تأطير شرعي + أثر اجتماعي + منهجية واضحة |
| مستثمر | حلم بصري + جدوى مالية + ROI |

### المرحلة ١ — الاستكشاف والبحث العميق `/research`

**الفلسفة:** مو مجرد جمع بيانات — تحليل واستنتاج وربط

**4 طبقات بحث:**

```
┌─────────────────────────────────────────────────┐
│  الطبقة ١: الأرقام الكبرى                       │
│  حجم السوق + نمو سنوي + توجهات + رؤية 2030     │
│  الاستنتاج: "الفرصة هي ___ لأن ___"             │
├─────────────────────────────────────────────────┤
│  الطبقة ٢: السلوك والثقافة                      │
│  أنماط الجمهور + دوافع نفسية + احتياجات + ثقافة │
│  الاستنتاج: "البصيرة الإنسانية هي: ___"         │
├─────────────────────────────────────────────────┤
│  الطبقة ٣: المنافسة المحلية                     │
│  خريطة منافسين (5-10) + نقاط قوة/ضعف + فجوات   │
│  الاستنتاج: "المساحة البيضاء هي: ___"           │
├─────────────────────────────────────────────────┤
│  الطبقة ٤: النماذج العالمية                     │
│  3-5 نماذج ناجحة + 1 فاشل + دروس مستفادة       │
│  الاستنتاج: "الدرس الرئيسي: ___"               │
└─────────────────────────────────────────────────┘
```

**المخرجات:**
- 5 بصائر رئيسية (جملة واحدة قوية لكل بصيرة)
- خريطة فرص (مصفوفة 2×2: سهولة التنفيذ × حجم الأثر)
- توصية استراتيجية ("نوصي بأن يكون المشروع ___ لأن ___")
- 3-5 أسئلة مفتوحة تحتاج توضيح

### المرحلة ٢ — بناء المفهوم الإبداعي `/concept`

**الرحلة:** بصيرة → نموذج جوهري → 3 سيناريوهات → شلال تطبيقات

**الخطوة ١ — البصيرة الجوهرية:**
```
"[الجمهور] يشعر بـ [عاطفة/حاجة]
 لأن [سبب عميق]،
 لكن [الحاجز الحالي] يمنعهم من [ما يريدونه حقاً]"
```

**الخطوة ٢ — النموذج الجوهري:**
| العنصر | الوصف |
|--------|-------|
| الاسم | يحمل المعنى بدون شرح |
| الغاية العليا | لماذا هذا المشروع مهم |
| جوهر المعنى | جملة واحدة تختصر كل شيء |
| المنطق الداخلي | لماذا هذا الحل وليس غيره |
| المبادئ التصميمية | 3-5 قواعد تحكم كل قرار |

**الخطوة ٣ — ثلاثة سيناريوهات (دائماً 3):**

| | 🔴 جريء | 🟡 ذكي (مُوصى) | 🟢 آمن |
|--|---------|----------------|--------|
| الطموح | أعلى ما يمكن | توازن مثالي | الأكثر عملية |
| المخاطر | عالية | محسوبة | أدنى |
| التكلفة | الأعلى | معقولة | الأقل |
| لمن | يريد يكون "الأول" | يريد أفضل قيمة | محافظ أو تجريبي |

**الخطوة ٤ — شلال التطبيقات:**
```
                    الفكرة الكبرى
                        │
           ┌────────────┼────────────┐
           ▼            ▼            ▼
      تجارب ميدانية  محتوى رقمي  منتجات/خدمات
      فعاليات        موقع         تطبيق
      ورش عمل       فيديوهات      منتج مادي
      تنشيط حضري    حملات سوشال   خدمة جديدة
```

لكل تطبيق: ماذا؟ لماذا؟ لمن؟ كيف يتفاعلون؟ كيف نقيس؟

### المرحلة ٣ — التخطيط والتنفيذ `/pricing`

**معادلة التسعير الموحدة:**
```
سعر البيع = التكلفة الفعلية × معامل القيمة × معامل العميل × معامل الإلحاح
```

**التكلفة الفعلية:**
```
التكلفة = تكلفة الفريق + تكلفة الأدوات + تكلفة الإنتاج + تكلفة أطراف ثالثة
```

**أسعار الفريق بالساعة:**
| الدور | ريال/ساعة |
|-------|-----------|
| مدير إبداعي | 350 |
| مدير مشروع | 250 |
| استراتيجي/باحث | 250 |
| مصمم أول | 200 |
| مصمم مبتدئ | 150 |
| كاتب محتوى | 150 |
| مطور (ويب/تطبيقات) | 200 |
| مصور/مخرج فيديو | 250 |
| منسق تنفيذ | 120 |

**المعاملات:**

| المعامل | القيمة | الحالة |
|---------|--------|--------|
| **معامل القيمة** | 1.5x - 2.0x | عمل تشغيلي عادي |
| | 2.0x - 3.0x | عمل إبداعي (تصميم، محتوى) |
| | 3.0x - 5.0x | عمل استراتيجي (استشارات) |
| | 4.0x - 8.0x | عمل تحولي (تحول مؤسسي) |
| **معامل العميل** | 0.8x | شركة ناشئة/صغيرة |
| | 1.0x | شركة متوسطة |
| | 1.2x | شركة كبرى |
| | 1.3x - 1.5x | جهة حكومية/سيادية |
| **معامل الإلحاح** | 1.0x | جدول زمني عادي |
| | 1.3x | مستعجل (أقل 30% من المعتاد) |
| | 1.5x - 2.0x | طوارئ (أقل من أسبوع) |

**3 مخرجات للتسعير:**
1. **عرض العميل** — تفصيل منطقي ومبرر (بحث + تصميم + إنتاج + إدارة + احتياطي + ضريبة)
2. **لوحة الربحية الداخلية** — إيراد + تكلفة + صافي ربح + هامش% + عائد/ساعة
3. **تبرير السعر** — 3-5 أسباب لماذا السعر عادل
